# Express

A better version of [hideout](https://github.com/hikalg/hideout), with actually functional buttons.

## URLs

- https://hikamarine.au
- https://hikalg.github.io/express

## Dependencies

- Astro
- Google Sans Flex font (for the best experience)

## Installation
- `git clone https://github.com/hikalg/express.git`
- `cd express`
- `npm install` to install all dependencies
- `npm run dev` to launch the development server at `http://localhost:4321/express`
